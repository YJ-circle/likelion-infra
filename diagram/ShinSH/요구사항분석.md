# 요구사항 분석
1. 별점
- 1점에서 5점만 평가 가능
- 추가로 새로운 평가 가능 (한 집에 중복 평가 가능)
- 별점 수정 기능 (재평가 시스템)
- 별점과 리뷰 같이 입력해야 작성 가능
- 6점 이상의 별점은 작성 불가 알림 띄우기
- 별점 및 리뷰 삭제 기능
- 별점 소수점을 허용할지???

2. 리뷰
- 리뷰 수정 및 삭제 가능
- 리뷰 작성 글자수는 최소 10자 이상
- 리뷰 작성 글자수 최대 자 수는??
- 리뷰에 이미지도 첨부 가능한지??

3. 랭킹시스템
- 맛집별 평균 별점으로 순위 산정
- 평균 별점이 높은 맛집 부터 순위 산정
- 별점 수정, 삭제 및 새 평가 시에 평균 별점 다시 계산 후 순위 산정
- 한 맛집에 대한 사용자의 가장 최근 평가로 순위 산정 (중복평가 적용 안됨)
- 평가 없는 집은 랭킹에서 제외하기
- 랭킹은 몇위까지 나오게 할것이지??
